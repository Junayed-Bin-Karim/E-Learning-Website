Course Overview

Machine Learning is reshaping the world. This intensive 3-month bootcamp is designed to take you from writing your first line of Python code to deploying complex Machine Learning models. You will work with real-world datasets, clean messy data, visualize trends, and predict outcomes using powerful algorithms like Regression, SVM, and Neural Networks.

What You Will Master
Python for Data Science

Master libraries like Pandas, NumPy & Matplotlib.

Supervised Learning

Linear Regression, Logistic Regression, SVM.

Unsupervised Learning

K-Means Clustering, PCA, and dimensionality reduction.

Model Deployment

Deploy models using Flask/Streamlit for the world to see.

Curriculum

12 Weeks

  • Python Syntax, Data Types & Functions
  • High-Performance Computing with NumPy
  • Data Manipulation & Cleaning with Pandas
  • Data Visualization (Matplotlib & Seaborn)
  • Capstone 1: Exploratory Data Analysis (EDA) on Titanic Dataset

  • Math for ML: Linear Algebra & Statistics

  • Intro to Neural Networks & Deep Learning
  • Model Evaluation: Precision, Recall, F1-Score
  • Deploying Models with Streamlit & Flask
  • Final Project: End-to-End Disease Prediction Web App
